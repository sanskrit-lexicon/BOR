# Changelog

All notable changes to this repository's metadata and documentation are recorded
here. Format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [v1.0-triage] - 2026-05-30

### Added
- Full GitHub issue taxonomy: exactly one type label, one severity label, and
  one milestone per issue; each issue assigned to its matching org project.
- Expanded `README.md`: contents, timeline, live Projects & Milestones table with
  Mermaid charts, issue typology, source bibliography, encoding declaration, and a
  pipeline diagram.
- Expanded `CLAUDE.md`: architecture, correction commands, and a data-format
  reference with an annotated example entry.
- Community health files: `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`
  (Contributor Covenant 2.1), `SECURITY.md`.
- `LICENSE` (CC BY-SA 4.0) and this `CHANGELOG.md`.
